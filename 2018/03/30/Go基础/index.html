Go的package用法

  1. Go每个文件必须要有包的声明，并且必须在第一行，比如：package main
  2. 同一个路径下，只能存在一个package，一个package可以拆成多个源文件
  3. 如果要生成Go可执行程序，必须要有main的package包，且必须在该包下有main函数

Go的import用法

  2. 不允许导入源文件没有用到的包，否则编译器会报错（这个是不是比java .net强多了 哈哈）

Go的数据类型

  1. 整型：unit8(0-255) unit16(0-65536) unit32(0-4294967295) unit64(0-太大了懒得写了) int8(-128-127) int16（-32768-32767） int32 int64（太大了懒得写了）
  2. 浮点型：float32 float64
  3. 复数类型：complex64 complex128
  4. 布尔类型： bool ==>true or false
  5. int所占的字节取决于系统位数，32系统位占4个字节，64位系统占8个字节
  6. 字符串类型：Go语言的字符串的字节使用UTF-8编码标识Unicode文本。

Go的变量声明

  1. var 变量名称 变量数据类型 = 赋值
  2. var 变量名称 变量数据类型
  3. 变量名称 = 赋值
  4. 变量名 := 赋值

  5. 分组声明

  6. 声明多个变量,”var a,b,c int = 1,2,3”

  7. 全局变量必须使用var关键词，局部变量可以省略
  8. ““ 下划线这个另类的特殊变量，只要赋值给，表示已经活到头了，over了，被回收了
  9. Go类型转换格式：变量名称 [ : ]= 目标数据类型 (需要转换的变量)
  10. 切记：Go中不存在隐式转换，类型转换必须是显示转换

重点说明一下go里声明变量数据类型继承最近的变量数据类型

Go的变量作用域

  1. 在函数体内声明的变量称为局部变量，并且只在函数体内有效
  2. 全局变量，定义在函数外的变量，且必须使用var关键字。全局变量在该包名下所有位置有效。

Go的常量声明

  1. 显示声明：const 变量名称 变量数据类 = 赋值
  2. 隐式声明：const 变量名称 = 赋值
  3. 常量定义可以使用内置表达式定义，比如 len() ,unsafe.SizeOf()等等，但是呢，是不支持自己写的表达式滴
  4. 常量目前支持布尔类型、数字类型、字符串类型

另类的常量 iota

  1. iota在你const关键字出现时将被重置为0
  2. const中每新增一行常量声明将使iota计数+1

Go里的函数

  1. 声明格式 func 函数名称 (参数1，参数2，…) 返回类型 { 函数体 }

Go的数组

  1. 声明方式：var 数组名称 [大小] 数组类型，eg：var array [6] int
  2. 数组初始化：var array [2] int {1,2}
  3. 数组访问：数组名称[下标]，eg: array[1] 得到的就是1

Go的多维数组

  1. 访问数组:array[0][1]

Go里的动态数组-切片（对数组的抽象）

  1. 声明方式: var 变量名称 [] 数据类型（注意这里一定不设置长度，否则就是定长的数组啦）
  2. 使用make方法来创建切片，var 变量名称 [] 数据类型 = make([] 数据类型,长度) eg:var slice1 []int = make([]int,10)
  3. 初始化切片s,是数组arr的引用: var array =[]int {1,2,3} slice2 := array[0:2]
  4. 获取长度len(),eg: len(slice2)
  5. 获取切片的容量，eg: cap(slice2)
  6. 切片截取：slice2[0:1]
  7. 切片追加新的元素：slice2=append(slice2,1,2)
  8. 切片复制：拷贝 slice1 的内容到 slice2 ==>>> copy(slice1,slice2)

Go里的range(range 关键字用于for循环中迭代数组(array)、切片(slice)、通道(channel)或集合(map)的元素)

Go的Map（无序的键值对的集合）

  1. 声明方式：var 变量名称 map[key的数据类型] 元素的数据类型

  2. 使用make函数创建：变量名称 := make(map[key的数据类型] 元素的数据类型)

  3. 删除元素：delete(maptest,”尼玛”)
